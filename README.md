# Metro Ticket Generating System – ServiceNow

## Project Progress

The Metro Ticket Generating System is developed using the ServiceNow platform to automate metro ticket booking, fare calculation, and ticket management.  
This project follows a structured phase-wise approach, covering planning, development, testing, deployment, and final presentation.

---

## Phase 1: Requirement Analysis & Planning

- Identified passenger and admin requirements
- Defined functional and non-functional requirements
- Prepared project scope, objectives, and deliverables
- Created requirement analysis and planning documentation
- Finalized project timeline and milestones

---

## Phase 2: Backend Development & Configurations

- Created scoped application in ServiceNow
- Designed custom tables for:
  - Metro Tickets
  - Stations
  - Routes and Fare Management
- Configured Service Catalog item for ticket booking
- Implemented automation using Flow Designer
- Enabled notifications and record creation logic

---

## Phase 3: UI/UX Development & Customization

### Interface Design
- User-friendly Service Catalog form
- Dropdown-based station selection
- Read-only auto-calculated fare field

### Navigation Flow
- Passenger → Service Portal → Ticket Booking
- Admin → Application Module → Ticket & Fare Management

### Usability
- Simple and intuitive design
- Reduced number of user inputs
- Clear confirmation messages after ticket generation

---

## Phase 4: Data Migration, Testing & Security

- Added sample metro stations and fare data
- Performed:
  - Functional Testing
  - System Testing
  - Performance Testing
  - User Acceptance Testing (UAT)
- Implemented role-based access control (RBAC)
- Configured ACLs for data security
- Ensured data accuracy and consistency

---

## Phase 5: Deployment, Documentation & Final Presentation

- Final validation of workflows and forms
- Deployment in ServiceNow Personal Developer Instance
- Prepared complete project documentation:
  - Ideation
  - Planning
  - Requirement Analysis
  - Design
  - Testing
  - Final Report
- Project demo and final presentation completed

---

## Technologies Used

- ServiceNow Platform
- Service Catalog
- Flow Designer
- Custom Tables
- Reports & Dashboards

---

## Project Conclusion

The **Metro Ticket Generating System using ServiceNow** successfully automates the metro ticket booking process and demonstrates the effective use of ServiceNow beyond IT service management. The system improves efficiency, reduces manual effort, and enhances user experience.

This project provided hands-on experience in ServiceNow application development, workflow automation, and real-world problem solving, fulfilling all academic and course requirements.
